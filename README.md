# tomadoro! 🍅

> a pixel pomodoro timer — grow your tomato, earn your break

🌱 → 🌿 → 🌺 → 🍅

---

## what is it?

Tomadoro is a 30-minute focus timer built with Electron. Instead of a boring countdown, you grow a little pixel tomato plant while you work. Stay focused, watch it bloom. Get distracted... well, the tomato won't judge you.

---

## growth stages

Your plant evolves every 5 minutes through 6 stages:

| stage | name | time |
|---|---|---|
| 🌱 | Tomato One | 0 – 5 min |
| 🌿 | Tomato Two | 6 – 11 min |
| 🪴 | Tomato Three | 12 – 17 min |
| 🌺 | Tomato Four | 18 – 23 min |
| 🍅 | Tomato Five | 24 – 29 min |
| ✨ | Tomato Six |    30 min  |

---

## how to run

Make sure you have **Node.js** installed, then:

```bash
cd tomadoro
npm install
npm start
```

---

## controls

| button | action |
|---|---|
| 🌱 tap seed | start the timer |
| ⏸ pause | freeze time |
| ▶ continue | keep growing |
| ⏻ quit | go home |
